Try it out
  • Download Sync Gateway from the Couchbase downloads page.
  • Start Sync Gateway from the command line. 
    $ ~/Downloads/couchbase-sync-gateway/bin/sync_gateway
  • Install CORS proxy to allow sending request from the browser. 
    npm install -g corsproxy
