  • Java Listener: Clone liteserv-java and run start it from the command line: ./gradlew runApp
  • macOS Listener: Download the LiteServ mac app packaged with Couchbase Lite macOS and start it: $ LiteServ
  • Install CORS proxy to send requests from the browser to the Couchbase Lite Listener. 
    npm install -g corsproxy
