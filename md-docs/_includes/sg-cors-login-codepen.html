Try it out
  • Download Sync Gateway from the Couchbase downloads page.
  • Create a new file named sync-gateway-config.json with the following. 
    {
  "log": ["HTTP", "Auth"],
  "CORS": {
    "Origin":["http://s.codepen.io"],
    "LoginOrigin":["http://s.codepen.io"],
    "Headers":["Content-Type"],
    "MaxAge": 1728000
  },
  "databases": {
    "sync_gateway": {
      "server": "walrus:",
      "users": { "GUEST": { "disabled": false, "admin_channels": ["*"] } }
    }
  }
}
  • Start Sync Gateway from the command line. 
    $ ~/Downloads/couchbase-sync-gateway/bin/sync_gateway sync-gateway-config.json
  • Install CORS proxy to send requests from the browser to Sync Gateway. 
    npm install -g corsproxy
corsproxy
